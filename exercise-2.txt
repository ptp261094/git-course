1.
- git commit --amend: go to vim mode to edit the latest commit message
- git commit --amend -m "<message>": update new commit message directly
- git restore --staged <file>: revert a file from Staging to Working Directory
- git reset HEAD~1.: revert 1 commit from Repository to Working Directory
